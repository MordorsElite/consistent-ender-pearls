# Consistent Ender Pearls

This mod removes the slight randomization of a thrown Ender Pearls momentum. So if you stand in the exact same spot and look in the exact same direction, the ender pearl will always land exactly the same.

To be clear, you will still have to account for the gravity-induced arc in your throw. The enderpear will not simply fly exactly where your crosshair was. 

It just removes the cases where you were trying to aim at a specific spot and the random momentum makes it so your pearl lands somewhere else.

## How to build the mod yourself:

Import it in your favorite IDE, wait for Java and Gradle to load, then build the mod with `./gradlew build`.
